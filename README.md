# Brainstorm App

### Setup
The following are procedures to follow when setting up the Brainstorm app for the first time.

```shell
# install packages
npm install

# set environment variables
cp .env.example .env
```

### Run the project

```shell
# watch mode
npm run start:dev

# production mode
npm run start:prod
```

### Run tests

```shell
# unit tests
npm run test

# e2e tests
npm run test:e2e
```
